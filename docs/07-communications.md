# 7. Communications

## 7.1 Messaging

The messaging features should be simple, and easy to use.

### 7.1.1 Overview

### 7.1.2 Inbox

The Inbox is where you can see all of your conversations in one place.  
You can also create new chats and edit them. 

<Image 
  src="/assets/communication/inbox.svg"
  alt="Inbox"
  caption="Inbox"
/>

The left sidebar has a [+ New Chat] button at the very top.

### 7.1.2.1 New Chat

To create a new chat, press the [+ New Chat] button.

The New Chat window pops up.

- Choose DM or Group Chat  
- Choose member(s)
- Create and Send Message
 
### 7.1.3 Direct Message

When you click the conversation from the list, the chat window for that conversation appears from the right.

<Image 
  src="/assets/communication/inbox-full.svg"
  alt="Inbox"
  caption="Inbox"
/>

Here you can continue the conversation, by using the textbox at the bottom.

You may also pin the conversation or mute notifications.

#### Message Box

You can enter a new message or reply to an existing comment.

You can upload a file

You can link ERP objects

### 7.1.4 Object Links

You can create button like links that takes the user to the corresponding object when clicked.

List of Objects that can be linked
- Purchase Request
- Invoice
- Bill

- Shipment
- Inventory Item
- Sensor Serial Detail
- Production Batch

- Project
- Task
- RMA


### 7.1.4 Group Chat

Same as Direct Message, but you can add multiple members.
